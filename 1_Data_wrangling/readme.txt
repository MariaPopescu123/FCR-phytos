1_Data_wrangling

Scripts to pull in data published at the Environmental Data Initiative (EDI) and format it for analysis.

List of files:

1A_HOX_EM_operation.R: script to create a schedule of hypolimnetic oxygenation (HOx) and epilimnetic mixing (EM) systemp operation in Falling Creek Reservoir                          

1B_Chemistry.R: script to download published water chemistry data from EDI and format it for analysis                                

1C_Cmax_WtrTemp_DO_pH_Stability_Thermo.Depth.R: script to download published depth profile data collected using a conductivity-temperature-depth (CTD) profiler, a YSI sonde, and a thermistor string deployed at FCR and format it for analysis

1D_PAR.R: script to download published photosynthetically active radiation (PAR) data and format it for analysis                                      

1E_Photic_zone_WtrTemp_DO_pH.R: script to download published depth profile data of water temperature, dissolved oxygen (DO) and pH and format it for analysis                

1F_Secchi.R: script to download published Secchi data and format it for analysis                                   

1G_FP_Distribution_metrics.R: script to download published depth profiles of phytoplankton fluorescence-based biomass data and format it for analysis                  

1H_Biodiversity_indices.R: script to download published phytoplankton biovolume data and calculate biodiversity indices for analysis                     

1I_Community_structure.R: script to download published phytoplankton biovolume data and calculate community structure indices for analysis                      

1J_Megamatrix.R: script to collate all physicochemical and phytoplankton variables into a single matrix                               

1K_WRT.R: script to download published Falling Creek Reservoir inflow stream discharge data and calculate water residence time (WRT)
